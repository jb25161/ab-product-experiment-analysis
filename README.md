# A/B Product Experiment Analysis

This project simulates an A/B test used in product management to evaluate whether a new feature improves user engagement.

## Objective
Determine if a new product feature increases user click-through rate compared to the existing feature.

## Experiment Design
- Simulated 1,000 users
- Split into two groups:
  - Control group (existing feature)
  - Test group (new feature)

## Methodology
The experiment uses:
- Python
- NumPy
- Pandas
- Statistical hypothesis testing (two-proportion z-test)

## Results
The simulated experiment shows the test feature produces a higher click-through rate than the control group.

Control CTR: ~22.6%  
Test CTR: ~25.8%
